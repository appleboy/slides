## 議程簡介

本場 90 分鐘工作坊，超越「用 AI 寫程式碼」的範疇，展示一個近乎全自動的流程——從開發、審查到部署——開發者僅需介入兩個關鍵點：初始方向設定與最終核准。

## 議程結構

### 前 20 分鐘：完整流程示範

- 自動程式碼生成
- PR 建立
- AI 程式碼審查
- 自我修正
- CI/CD 錯誤診斷

### 中間 60 分鐘：動手實作

- 計畫生成與迭代精煉
- AI 實作程式碼
- 程式碼優化
- 安全掃描
- PR 提交
- 迭代審查解決
- 自動化 Pipeline 修復

### 最後 10 分鐘：討論

- 針對不同團隊規模調整工作流程
- 工具鏈整合
- 透過自訂 AI Agent Skills 擴展覆蓋範圍

## 核心主題

- AI 生成執行計畫與迭代精煉
- 自動化程式碼重構與安全掃描
- AI 驅動的 PR 建立與程式碼審查循環
- 自我修復 CI/CD Pipeline（AI 讀取錯誤日誌、定位問題、修復直到 Pipeline 全部通過）
- 自訂 AI Agent Skills 擴展自動化範圍

## 前置需求

- 自備筆電
- 任意 AI 工具帳號（Claude Code、Cursor、GitHub Copilot 或類似工具）
- 基本 Git 操作能力與 CI/CD 概念理解
